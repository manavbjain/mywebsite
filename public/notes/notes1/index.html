JavaScript Tips I Wish I Knew Earlier

Here are some JavaScript tips that have made my development process much smoother.

1. Use Optional Chaining

Instead of writing multiple checks:

1
2
3

if (user && user.address && user.address.street) {
  console.log(user.address.street)
}

Use optional chaining:

1

console.log(user?.address?.street)

2. Destructuring with Default Values

You can provide default values when destructuring:

1

const { name = 'Anonymous', age = 0 } = user

3. Array.from() for Quick Array Generation

Generate arrays with specific patterns:

1
2
3
4
5

// Create array [0, 1, 2, 3, 4]
const numbers = Array.from({ length: 5 }, (_, i) => i)

// Create array of repeated values
const zeros = Array.from({ length: 3 }, () => 0) // [0, 0, 0]

4. Use Set for Unique Values

Remove duplicates from an array:

1

const uniqueValues = [...new Set([1, 2, 2, 3, 3, 4])] // [1, 2, 3, 4]

5. Object Property Shorthand

When the property name matches the variable name:

1
2
3
4
5
6
7
8

const name = 'John'
const age = 30

// Instead of
const person = { name: name, age: age }

// Use shorthand
const person = { name, age }

These small improvements can make a big difference in code readability and maintainability!
